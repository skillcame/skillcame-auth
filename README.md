## 🚀 Overview

**SkillCame** is a comprehensive AI-powered Learning Management System (LMS) built with React and Firebase. It provides a complete ecosystem for online learning with advanced features including:

- 🎓 Structured AI Courses with Progress Tracking
- 🤖 Multiple AI Chat Models Integration
- 📰 Real-time AI News & Industry Insights
- 💳 Integrated Payment System
- 👥 User & Admin Management
- 🔐 Role-based Access Control
- 📱 PWA Support with Offline Capabilities
- 🎨 Modern UI with Animations
- ⚡ Optimized Performance

**Live Demo:** https://skillcame.vercel.app
